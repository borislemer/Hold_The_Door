# Malicious IP lists

Generated by `pull_malicious_ips.py` from:

- **blocklist.de** – https://www.blocklist.de/downloads/export-ips_all.txt  
- **firehol** – https://github.com/firehol/blocklist-ipsets (all `.ipset` except blocklist_de* and bogons)  
- **bitwire** – https://raw.githubusercontent.com/bitwire-it/ipblocklist/master/inbound.txt  
- **cybercrime** – abuse.ch Feodo Tracker, cybercrime-tracker.net (fuckerz.php), firehol cybercrime.ipset  

## Naming

`<source>_<category>_<ddmmyyyy>_<extras_if_required>.txt`

- **Per-source IP lists:** `blocklist_de_ips_28022026.txt`, `firehol_ips_28022026.txt`, `bitwire_ips_28022026.txt`, `cybercrime_ips_28022026.txt` — one IPv4 per line, no size limit.
- **Combined ranges (CIDR):** `combined_ranges_28022026.txt` — minimal set of CIDR ranges covering all sources, one CIDR per line.

Raw URLs (once pushed):  
`https://raw.githubusercontent.com/<user>/<repo>/<branch>/lists/<filename>.txt`
